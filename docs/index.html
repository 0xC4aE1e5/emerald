What is blang?

blang is a programming language that is very minimal and does not need a lot of resources. It's built on top of Crystal. You can try it out by making a blang script called hworld.blang and put this in the file: 

say Hello, World!
and then compiling and running blang with hworld.blang as the argument. Also, blang handles errors differently. It's not like Python where it figures it out before running it and throwing an error, no. blang runs the code before the error, errors out on the line with an error, and then runs the next line like usual.
